About SteamGifts EasyEnter

Reason

This extension is made by Thomas de Luca.
I made the extension because I wanted to enter giveaways more quickly. I was fed up by clicking on every giveaway, then clicking on "Enter giveaway" and closing the tab.
Because I couldn't find any extensions that semi-automated this proces for me, I saw it as an opportunity to learn how to write a basic extension *and* to automate the proces of entering giveaways.
Now, this extension is far from perfect. But it does it's job well enough for me.

How it works

  1. First of all, activate this extension before you visit the Steamgifts website.
  2. Open giveaways in new tabs. The extension automatically enters the giveaways for you.
  3. When you run out of points the extension alerts you with a pop-up.
  4. Click on the extension icon and then on "Close ALL Giveaway-tabs". This will close all the open giveaway tabs.
  5. ???
  6. Profit!
If there are 0 open giveaway tabs, the button shows "Visit Steamgifts website".
If there are open giveaway tabs it shows "Close ALL open giveaway-tabs".

Security

This app does *not* collect any user info. Don't worry about your steamaccount, it's safe since I'm not that good at coding extensions.
If you're a suspicious type, open the files of this extension and take a look at the souce code.
